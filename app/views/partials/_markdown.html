You can use markdown in this field

Markdown is a way to format text for the web, that doesn't require you to write HTML code.

To write a header, use two hash symbols and a space, for example: ## Heading.

For sub headings of a heading, use 3 hash symbols.

Never use a single hash to define a header.

To add a link, use square brackets for the text and round brackets for the URL, for example: [GOV.UK](https://www.gov.uk).

To add a list, use a hyphen followed by a space, for example: - list item 1 and put each list item on a new line.

To add a numbered list, use a number followed by a full stop and a space, for example: 1. list item 1 and put each list item on a new line.
